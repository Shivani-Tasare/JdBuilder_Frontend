FAQS

Which browser JD Supports

Smart supports both Google Chrome and IE-11, However for best user experience use Google Chrome

What is "Make it as Private JD"

Select this option, if you want to have your JD as private and not visible to others during search

What is Mandatory/Desired Tags

To create a JD, Mandatory, Desired Tags are necessary to add skills that are required for that designation. By having so, mandatory and desired skills can be added under mandatory, desired skills section during JD creation.

Login
Please navigate to the application using the URL https://jdbuilderweb.azurewebsites.net/. If you are doing Login for the first time, you would be prompted to enter your DMI Active Directory credentials – the same that you use for Microsoft Outlook and post authentication, you would be directed to Dashboard/Main page. If you have done Login in past, the system may directly take you to Dashboard page.
Dashboard
There are several things you can do from the Dashboard: Search by Keyword Desgination Experience Created by
Search by Keyword
Basic keyword search can be achieved with this option, which will bring JD’s which containing the keyword given in search column.
Designation
JD’s Designation can be selected from the drop down list and view.
Experience
Experience drop down option have list of “Years of Experience”, by selecting one of the listed values from the drop down will display the JD’s with matches the selected value.
Created By
Created By drop down option have list of “Names” who have created the JD’s sofar, by selecting one of the listed values from the drop down will display the JD’s with matches the selected value.
Location
Location drop down option have list of “Location” from which JD’s are created, by selecting one of the listed values from the drop down will display the JD’s with matches the selected value.
Create New JD
To Create New JD, click on the button “Create New JD’s” and fill the appropriate columns in the form to create new JD. Refer “Create New JD” video for more details.
Edit JD
To Edit existing JD’s content use this option and this can be achieved by selecting the JD from dashboard Dashboard – Select JD -> View Details -> Edit (at bottom of the page) Refer Edit JD video for more details
